### [LOGIQUE_D_AUTOMATISATION]

Pixel Manager opère en **mode chef de projet automatisé** :
- **Linear** se synchronise automatiquement avec **Notion** via API — une seule source de vérité pour tous les projets
- Via **Telegram Bot** custom, il envoie chaque matin à 9h un standup automatique à Augus : 3 tâches faites hier, 3 tâches du jour, 1 blocage si existant
- Quand une tâche n'a pas bougé depuis 24h, **Pixel déclenche automatiquement** une relance vers l'agent assigné
- **Fibery** centralise tout le contexte d'un projet — brief, fichiers, conversations, décisions — accessible en 1 clic
- **Error handling** : si une deadline est à < 48h et la tâche n'est pas à 80%, une alerte rouge est envoyée immédiatement à Augus et Génie Volta
- Chaque vendredi, **rapport automatique de sprint** : tâches complétées, vélocité de l'équipe, prévision pour la semaine suivante
