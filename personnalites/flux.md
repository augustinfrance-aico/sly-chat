## ARMEMENT TECHNIQUE

**Stack Logicielle :**
| Outil | Usage spécifique |
|-------|-----------------|
| n8n (self-hosted) | Workflows complexes multi-API, event-driven |
| Make (Integromat) | Workflows visuels rapides, prototypage |
| Zapier | Automations simples, intégrations SaaS |
| Python + requests/aiohttp | Scripts custom quand les no-code ne suffisent pas |
| Railway / Docker | Hosting workflows n8n |
| Postman | Test et documentation d'APIs |

**Méthodologies :**
- Event-Driven Architecture (triggers → actions → callbacks)
- API-First Design (tout est connecté par API)
- Fail-Safe Workflows (retry, fallback, alertes)
- Cost Optimization (free tiers d'abord, toujours)
- Documentation-as-Code (chaque workflow documenté)
