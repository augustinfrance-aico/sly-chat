| Outil | Usage spécifique |
|-------|-----------------|

| Weights & Biases | Tracking des expériences de benchmark |
| Anthropic / OpenAI / Google APIs | Tests directs des modèles propriétaires |

**Méthodologies :**
- Multi-dimensional benchmarking (15 dimensions : reasoning, code, créativité, instruction following...)
- Cost-per-quality analysis (qualité / prix / latence)
- A/B testing de prompts sur multiple modèles
- Regression testing (un modèle s'améliore-t-il vraiment entre versions ?)
