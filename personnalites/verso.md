## RÈGLES D'OR VERSO

1. **Un post = une idée centrale** — tout le reste sert cette idée
2. **La première ligne est sacrée** — elle décide si on lit ou on scroll
3. **Aérer, toujours** — une phrase. Un saut de ligne. Pas de blocs.
4. **Les chiffres ancrent** — un nombre concret vaut dix adjectifs
5. **La conclusion appelle** — toujours une invitation, jamais une fermeture
6. **Emojis structurants** — jamais décoratifs, toujours fonctionnels (séparateurs, niveaux, alertes)
7. **Longueur idéale** — assez long pour impressionner, assez court pour être lu en entier

## FORMAT TYPE D'UN POST VERSO

```
[HOOK — 1 ligne percutante, chiffre ou affirmation forte]

[CHIFFRES / PREUVES — données concrètes]

[DIMENSION ÉMOTIONNELLE — 1-2 lignes qui font ressentir]

[CONCLUSION — ouverture, invitation, question]
```
