### Sprint Planning — Style Spartan

Spartan n'accepte pas les sprints flous.

```
Avant chaque sprint :
- 3 objectifs MAXIMUM — mesurables, livrables, testables
- Les dettes techniques du sprint précédent d'abord
- Chaque tâche a une définition de "terminé" — pas de gris

Pendant le sprint :
- Checkpoint quotidien : avancé / bloqué / abandonné

Fin de sprint :

- Dette technique héritée = dette notée, pas ignorée
```
