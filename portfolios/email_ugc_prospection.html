Guide — Scraper des marques cibles

  1. Instagram — Chercher hashtags niche (#streetwearfrance, #skincareroutine, #newbrand). Noter les comptes avec 1K-50K followers (sweet spot = pas d'agence, besoin de contenu).
  2. Shopify Store List — Utiliser myip.ms ou store.leads pour trouver des boutiques Shopify par catégorie. Export CSV.
  3. TikTok Shop — Explorer les marques émergentes sur TikTok Shop. Cibles = celles avec produits mais peu de UGC.
  4. Apollo.io / Hunter.io — Trouver les emails des fondateurs / marketing managers. Gratuit jusqu'à 50 contacts/mois.
  5. Enrichir le questionnaire — Avant l'email, visiter le site de la marque et pré-remplir : {PRODUIT_STAR}, {SECTEUR}, {PLATEFORME}.

Variables à personnaliser

  1. {PRENOM} — Prénom du fondateur / marketing manager
  2. {MARQUE} — Nom de la marque
  3. {PRODUIT_STAR} — Le produit phare (trouvé sur leur site/IG)
  4. {SECTEUR} — Mode, beauté, tech, food, fitness...
  5. {PLATEFORME} — Où vous les avez trouvés (Instagram, TikTok...)

Tips Prospection

  • Sujet email court — max 6 mots, le prénom + la marque = personnalisation immédiate
  • CTA = lien vers un exemple — générer 1 visuel gratuit avec leur produit AVANT d'envoyer = taux de réponse x3
  • Envoyer mardi-jeudi 9h-11h — meilleur taux d'ouverture pour les e-commerçants
  • Volume — 30-50 emails/jour max pour éviter le spam. Utiliser un domaine secondaire
  • Follow-up automatique — Utiliser Instantly.ai ou Lemlist (plan gratuit dispo) pour automatiser la séquence 3 emails
  • Landing page = portfolio — Le lien dans l'email pointe vers portfolio_ugc_ia.html avec le questionnaire
  • Scraper → Questionnaire → Avatar → Email avec démo — le pipeline complet tient en 30 min par marque
