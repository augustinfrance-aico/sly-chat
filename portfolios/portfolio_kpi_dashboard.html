The 4 sheets I'll build

  • CSV Import — You paste your monthly export here. That's it. One action.
  • Master Database — Permanent storage. Grows month after month. Never breaks.
  • Monthly View — Formatted table per month, per agent. Auto-generated from database.
  • KPI Dashboard — All calculations done. Replacements/100 chats, refund rate, SLA %, cost/conversation — everything.

📈
KPIs auto-calculated

  • Replacements per 100 chats — (faulty + warranty + incorrect) / chats * 100
  • Gifts $ per 100 chats — gifts total / chats * 100
  • Reopened tickets % — reopened / total conversations
  • Follow-ups required % — follow-ups / total conversations
  • First response within SLA % — weighted average across agents
  • Resolution within SLA % — weighted average across agents
  • Cost per conversation — total cost / total conversations
  • Refund rate % — refund count / total conversations

⚙
Technical approach

  • Apps Script onEdit trigger — Detects when CSV is pasted, auto-routes to database
  • Named ranges — Clean architecture, no scattered formulas
  • QUERY + ARRAYFORMULA — Monthly view pulls from master automatically
  • Weighted averages — Correctly calculated (not simple averages)
  • Scalable to 12+ months — Add data forever, nothing breaks
  • Zero manual formulas — You never type a formula. Ever.
