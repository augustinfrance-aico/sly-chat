Capabilities

What This Automation Does

🔗

Zakeke API Integration

OAuth 2.0 authentication, orders retrieval, print-ready file downloads. Handles pagination and rate limits.

📐

Template Mapping

Your PSD/INDD template stays intact. Script maps each order to the correct layer positions, sizes, and text fields.

🔤

Dynamic Text Styling

Applies your template's text styles — fonts, sizes, colors — to order metadata (name, size, color, order #).

📦

Batch Processing

Process 10, 50, or 200 orders at once. Each batch generates organized production sheets automatically.

🎯

Custom Layout

Not a simple grid — follows your specific production layout with precise placement per your template design.

🔄

Easy Template Updates

Change the template? Just update the PSD/INDD file. The script adapts — no code changes needed.
