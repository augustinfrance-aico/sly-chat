TITAN est ton bot Telegram IA personnel. Il comprend le langage naturel, les commandes slash, et les messages vocaux.

35 — Under the Hood

Architecture

Comment TITAN fonctionne sous le capot. Zero cout, performance maximale.

⚙ Stack Technique

Message Telegram | v Polling Loop (async, anti-doublon par message_id) | v Router (slash command ou langage naturel) | +---------+---------+---------+ | | | | Brain Voice Module Jacques | | | | v v v v Groq Cascade (6 modeles, quotas separes) | llama-3.3-70b -> llama-4-maverick -> llama-4-scout -> kimi-k2 -> qwen3-32b -> llama-3.1-8b | v (si tous epuises) Gemini Fallback (Google, gratuit) | v Response -> Telegram -> Toi
Cout mensuel : 0€
Groq free tier (6 modeles en cascade), Gemini free, Groq Whisper free, gTTS free, Render free tier.
Lazy Loading
50+ modules charges uniquement a la premiere utilisation. Boot rapide, memoire minimale.
Anti-doublon
Set de message_id (max 500, purge a 200). Lock asyncio pour empecher le traitement parallele.
Deploiement
Render (auto-deploy GitHub). Polling (pas webhook) pour simplifier.
