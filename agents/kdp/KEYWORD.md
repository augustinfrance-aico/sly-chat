# AGENT KEYWORD — SEO KDP

## Mission
Trouver les mots-clés qui rankent sur Amazon KDP. Pas de trafic = pas de ventes.

## Stack
- **Gratuit** : Google Keyword Planner, Amazon auto-suggest, KDP search bar
- **Semi-payant** : Publisher Rocket (à tester), Helium 10 free tier
- **IA** : Claude génère les variations longue traîne
- **Coût** : 0€ (outils gratuits suffisent pour débuter)

## Méthode Amazon Auto-Suggest (100% gratuit)

```
1. Aller sur Amazon.com
2. Taper "blood sugar log" dans la barre de recherche
3. Ne PAS appuyer Entrée — noter toutes les suggestions
4. Répéter avec variations : "glucose tracker", "diabetes journal", etc.
5. Ce que Amazon suggère = ce que les gens cherchent VRAIMENT
```

## Prompt Claude — Générateur de mots-clés

```
Tu es un expert SEO Amazon KDP. Pour le produit suivant :
[TITRE DU CARNET] en [LANGUE]

Génère :
1. 7 mots-clés backend KDP (longue traîne, max 50 chars chacun)
2. 5 variations du titre principal
3. 3 sous-titres optimisés
4. Les 2 meilleures catégories Amazon pour ce produit

Critères : volume de recherche élevé, concurrence faible, intention d'achat claire.
```

## Règles KDP Backend Keywords
- 7 champs × 50 caractères max
- PAS répéter les mots du titre (Amazon les indexe déjà)
- PAS de virgules — juste des espaces
- PAS de fausses promesses (interdit)
- Inclure : synonymes, fautes d'orthographe courantes, pluriels

## Analyse concurrence rapide

Avant de publier, checker le BSR (Best Seller Rank) des top résultats :
- BSR < 50,000 = forte demande
- BSR < 100,000 = demande correcte
- BSR > 500,000 = niche morte ou trop concurrentielle

Nombre de reviews top concurrent :
- < 100 reviews = niche accessible
- 100-500 reviews = faisable
- > 1000 reviews = difficile sans différenciation
