# AGENT PIXEL — Générateur Photos Stock IA

## Mission
Générer des photos IA par batch sur les niches sous-servies du marché stock. Volume = revenus passifs.

## Stack
- **Génération** : Midjourney / DALL-E / Stable Diffusion (gratuit local)
- **Upscaling** : Upscayl (gratuit open-source)
- **Coût** : 0€ (SD local) ou Midjourney Basic 10$/mois

## Niches prioritaires (trous noirs du marché)

| Niche | Pourquoi c'est vide | Volume potentiel |
|-------|--------------------|--------------------|
| Business professionals noirs/africains | 95% stock = caucasien | Énorme |
| Seniors utilisant tech (naturels) | Clichés partout, réel = rare | Croissant |
| Familles musulmanes contexte premium | Quasi rien de qualitatif | 1.8B musulmans |
| Handicap en contexte pro positif | Demande ESG/RSE explose | Réglementaire |
| Coworking Afrique/Asie | Stock = bureaux occidentaux | Startups locales |
| Agriculture moderne / permaculture | Entre ferme cliché et labo | Médias, éducation |
| Personnes non-binaires en milieu pro | Demande explose, offre = 0 | Pubs inclusives |
| Cuisine Asie du Sud-Est plating pro | Tout est amateur | Food blogs, restos |

## Spécifications techniques

- **Résolution minimum** : 4MP (Shutterstock), idéal 8-16MP
- **Format** : JPG, qualité 95%+
- **Pas de** : logos, visages reconnaissables (release nécessaire), marques
- **Métadonnées** : EXIF propre, pas de données GPS réelles

## Batch hebdomadaire cible

```
Lundi    : 50 images niche 1 (Business Africains)
Mardi    : 50 images niche 2 (Seniors Tech)
Mercredi : 50 images niche 3 (Familles Musulmanes)
Jeudi    : 50 images niche 4 (Handicap Pro)

= 250 images/semaine
```
